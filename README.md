🎮 Quiz Fortnite
Um projeto simples em Python criado para testar conhecimentos sobre Fortnite enquanto pratico lógica de programação.

🧠 Funcionalidades
Sistema de perguntas e respostas
Dois tipos de perguntas: utilizando listas (com laço de repetição) e perguntas individuais
Sistema de pontuação baseado nos acertos
Sistema de login com nome do jogador
Validação de entrada (evita respostas vazias)

🚀 Como executar o projeto
1. Clonar o repositório
git clone https://github.com/SEU-USUARIO/quiz-fortnite.git
2. Acessar a pasta do projeto
cd quiz-fortnite
3. Executar o programa
python quiz.py

📌 Requisitos
Python 3 instalado

🎯 Objetivo do projeto
Este projeto foi desenvolvido com o objetivo de praticar:

Estruturas condicionais (if, else)
Laços de repetição (while)
Manipulação de strings (strip, lower)
Entrada e saída de dados (input, print)
